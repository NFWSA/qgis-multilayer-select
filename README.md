Multilayer Select (QGIS plugin)
===
![Icon](./icon.png)

Add new selection tools that work across layers.

- Select by click
- Select by Rectangle
- Select by Radius
- Select by freehand
- Add/Remove to/from selection when pressing Shift/Ctrl
- Select all features from all layers
- Invert all features
